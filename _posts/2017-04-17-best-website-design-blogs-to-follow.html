Being a Web designer is all about expanding your knowledge and seeing what is being done in the latest examples of our profession. It's for that reason I put together this list for the best tips, advice, freebies and tutorials I've collected over the years.

1. CSS Tricks

Tips, Tricks, and Techniques on using Cascading Style Sheets.

2. Codepen

Demo or it didn't happen. Codepen is great for finding new examples for super cool web design demos.

3. Tutorialzine

Tutorialzine is the one place for high-quality web development tutorials. Learn cutting edge techniques in web development and design, download open source components and participate in the community.

4. Codrops

Codrops is a web design and development blog that publishes articles and tutorials about the latest web trends, techniques and new possibilities.

5. Scotch

Scotch is a web development blog discussing all things programming, development, web, and life.

6. The coding love

This one is here to make you smile, laugh and just generally have a jolly good time.

7. UX Reactions

Erm, today's word is procrastination.

8. Smashing magazine

Smashing Magazine, an online magazine for professional web designers and developers.

9. CSS author

Web/Mobile design & development blog, providing user interface design freebies, resources, articles, tools and more!

10. David Walsh Blog

A blog featuring tutorials about JavaScript, HTML5, AJAX, PHP, CSS, WordPress, and everything else development.

11. Commitstrip

This one is for all the GIT users out there.

12. Designmodo

Designmodo publishes web design articles, tutorials and has a great shop with the best WordPress and Bootstrap themes, plugins, UI Kits and more.
